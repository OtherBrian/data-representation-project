You may have noticed that the navigation bar has changed, and you have a few more options here.
The intention is for this area to the be the admin of the store, where users can manage the products, customers and orders.
Let's introduce each section one by one.

The Customers Page

Here you'll be able to view, create, update and delete customers from the store.
This is all done through the use of AJAX, and performs API calls to a MySql database that holds the customer data.

The Orders Page

Here you'll be able to view and delete the orders on the store.
This is all done through the use of AJAX, however these orders are not stored on a local database.
They are in fact being served from Shopify, the popular ecommerce platform. I have created a dummy Shopify store to which I can make API calls and create what are known as Draft Orders via the Draft Order API.

If you're curious, I had to use the Draft Order API rather than the usual Orders API because my products are not hosted on Shopify. The Orders API requires Shopify Product IDs, which my products do not have, so I've passed them as custom products to the Draft Orders API.

The Create Order Page

This page is a combination of the Products and Customers pages from before.
The user can first select a product (they can also create, update and delete existing products here), then select a customer.
Once they are happy with their selections, they can hit the Create Order button, and it'll make a POST request to my Shopify store's Draft Orders API and hopefully create the order. The user can then return to the Orders Page and see the order there.

Logout

This will simply log the user out and return them to the index page for anonymous users.
